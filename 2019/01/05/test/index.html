#

无问西东

我们每一个人都有可能是李想，不完美，却不断地在成长，越成熟，越能遵从内心的选择。

莎士比亚说：人生有如一块用善与恶的丝线交织成的布，我们的善行必须受到我们过失的鞭挞，我们的罪恶却又依赖我们的善行把它掩盖。

他在训练期间开着战斗机去废墟上给孤儿投递食物，受罚时他毅然地说：这个世界不缺完美的人，缺的是用内心发出的真心、正义、无畏，和同情。纵然受罚，他依然义无反顾，继续空投食物，还影响了一大批飞行学员加入。善意以星火燎原之势得以传播，让包括陈鹏在内的一大批云南孤儿得以存活。在沈光耀短暂的青春里，他面临着几个关键的选择，家与国，忠与孝，命与义。这些选择，没有标准答案，每一次都是两难的取舍。沈光耀的选择，永远是遵从他内心最真实的感受。真实，是他留给这个世界的姿态。
