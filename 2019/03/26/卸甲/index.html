卸甲

发表于 2019-03-26 | 更新于 2019-04-30 | 分类于 人生

昨天收到消息，批复下来了，那一刻，心情很复杂。原来一直盼着这一天，当这一天真的到来是这样的心情。三年来的时光迅速在脑海里掠过，像放电影一样。想起了刚报到时的满怀希望、激动的心情，那个时候真可谓是无忧无虑，每天可以干自己想干的事情，那个时候还在打游戏呢，LOL、吃鸡，看着大学同学们一个个都弃坑了，只有我还在每天玩着无聊的游戏打发时间，原来工作后，除了我大家都开始为生活忙碌起来了。再后来不知怎的我也不爱玩了，可能是因为少了一起玩游戏的人所以也就没什么意思了。17年那个时候还特喜欢游泳，以至于到现在微信名还是游泳呢。那个夏天，不能说是每天，但是一周总能去游个几次。蹬上共享单车，再塞一副耳机，整个世界都是我的。记得那时喜欢于文文，就听着《trouble is a friend》伴随着日落就出发了。不光喜欢游泳，还有去游泳的这段路。（人那，总是得寸进尺，得到了这个就还想要那个，永远不知道知足。谨以这句话先给我自己）再后来就去了另一个楼里工作，然后也开始逐渐忙碌起来。现在思绪看看那个时候的自己，就俩字：特么青涩！脸皮还是不够厚，可能是因为之前的人生太顺风顺水了，根本不需要厚着脸皮。本想来了单位能在前辈们的带领下学点东西为社会主义添砖加瓦呢，这里的确是学到了不少东西，只不过不是技术上的，人生呐，总是如此奇妙，你想要的没得到，得到的又不是你想要的，偏偏这个不是你想要的才是你最需要的。所以呢，这几年真的没有虚度啊，真的是学到了好多好多，足够我受用终生。我想如果毕业就去了地方，可能不会有现在的心智吧。其实还有挺多要学的，只不过我不能够再向大家继续学习了。最受用终生的就是养成了运动和自律向上的习惯吧，还有就是感谢大家教会我比以前更成熟了。在这里，我喜欢上了好多东西，各种运动，比如游泳、羽毛球、乒乓球、健身、甚至跑步，而且更爱篮球了呢。我曾跟朋友开玩笑说，我现在每天都运动，而且一周都不带重样的，嘿嘿，感谢你，让我变得越来越好。然后还喜欢三样东西，一是数学，一是音乐，还有英语。爱因斯坦和袁隆平小提琴也都拉的特棒。爱因斯坦说过科学和音乐其实是相得益彰相辅相成的。是这样的，音乐会提升你的感知能力，让你能从生命的灵性里对一个东西有一个感性的认识，这是非常难能可贵的。哦对了，在这里还有感谢我异常感性的室友立业同学，接下来就聊聊这里的每个人吧，排名不分先后奥。

立业同学，精致 Boy，感性。 非常感谢你这些年对我的容忍，虽然我经常看直播不带耳机。他活的非常精致，总是掏出一些我以前不知道但是很好用的玩意，处女座的书桌、衣柜，对自己东西放在哪了如指掌，各种小工具应有尽有，还喜欢“不务正业”做木匠，看到了明朝皇帝朱由校的影子，以致于清洁都常对他说，跟你一比我还真是个糙汉子。还有你对一件事的 intuition 真的是非常棒，一般我有事拿不准都会先听听你的看法，然后就觉得嗯对我怎么就没想到呢，你不仅教会了我从感性的角度认知一件事，还有对音乐的欣赏，我常看的唱歌主播，你一听就能知道她水平咋样，一开始我不以为然，后来仔细一听发现还真是。还有你的三观挺正的，也学到不少，但是还是有点担心你如此单纯以后能否应对这缤纷的世界啊哈哈。

露露，江湖人称露爷，原名张伟霆，因厌倦了娱乐圈的尔虞我诈故在此大隐隐于仕。怎么说呢，成熟稳重，任劳任怨，深刻诠释了“严于律己，宽以待人”八字箴言。给女票 5000 买个包眼睛眨都不带眨一下的，但是对自己总说自己穷，攒钱买 pad，对了，露爷的笔记本还是 i3 的呢，具体年限不明。而且露爷工作非常拼，加班可能是他的消遣方式，动不动“我上去一趟”，中午吃完饭遛弯到楼下也要上去加个班，晚上拿更是常态。值班从来不调休，除非女票有事。兢兢业业，克己奉公。

明灿，最有我大学同学影子的一位。随便骂随便损，不会斤斤计较，咋着都行，有点像大学里边互相叫儿子时候的感觉。在他身上我看到了一个人是多么复杂，一个宁肯花一万多请健身教练但是路上却还要为公交车在哪站下能省 5 毛钱合计合计的主，我是真的猜不透。为什么说明灿最像我大学同学呢，其实这是一个很高的评价，因为他们都很优秀，从他们身上我也学到了很多。明灿喜欢动漫，喜欢美国大片，喜欢摄影，爱好广泛且前卫。最主要的是脸皮厚，一个人比我们一整个屋子的人脸皮都厚，但这并不是坏事，脸皮厚点挺好的，像我这种脸皮薄的有的时候就很吃亏，很被动。至此 ，180club、上庄四少（不要脸的说：还有我）已全部介绍完毕。

清洁，活泼开朗，善解人意，人美心善大长腿，有着天使般的笑容，我们都亲切的称她“小天使”。小天使是庄里为数不多的女生，所以理所应当就成为了大家的团宠，大伙小心翼翼的百般讨好，四个单身汉（现在是三个）把无处安放的对对象的好全都施加在清洁一个人身上。小天使是最文艺的一个，满满一书架的书，平时喜欢看看音乐剧、话剧什么的，一张口就跟人感觉不是一般人，思想很深刻，考虑问题总能说到点上，这点我很佩服她。可能是因为我们四个对她的宠爱过于密集，导致工作这几年迟迟无人认领。但是，一回到学校，立马就坠入了爱河。

还有很多小伙伴，比如：不会踢球的不会弹琴的清哥、成熟稳重的彪哥、新进女儿奴川哥、思维敏捷的辉哥、幸福学姐孙心佩、两个小学妹画姐和莎姐、以及更小的小学弟刘斌张蒙。你们是我最不舍的上庄 50 号楼，过去的几年历历在目，一起吃火锅、包饺子、打麻将，当然还有加班。上庄的老陕都重新装修了，对面的店也换了一家又一家，不变的晋风依然是庄里最豪华的饭店。你们是我最不舍的上庄，在这里有我们共同的回忆，感谢你们在我的人生中留下浓墨重彩的一笔，也希望以后的我们依然对未来充满希望，不忘初心，砥砺前行，等以后我们再聚首时依然能够谈笑风生不辱使命。
